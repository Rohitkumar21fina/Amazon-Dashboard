# Amazon Dashboard

## Overview
This project is an interactive Power BI dashboard created using Amazon sales data. The dashboard helps analyze business performance through visual insights, KPIs, and data-driven analytics.

The project demonstrates skills in:
- Data Cleaning
- Data Visualization
- Business Intelligence
- Dashboard Design
- Data Analytics using Power BI

---

## Tools & Technologies Used
- Power BI
- CSV Dataset
- Microsoft Excel (if used for preprocessing)
- Data Visualization Techniques

## Project Files

### Dashboard File
- `POWERBI 8.pbix` → Main Power BI dashboard file

### Dataset
- `SECTION_C (1).csv` → Dataset used for analysis

## Dashboard Features
- Sales Performance Analysis

- Product Category Insights
- Customer Purchase Trends
- KPI Monitoring
- Interactive Filters & Visualizations
